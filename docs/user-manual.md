# 简历编辑器用户操作手册

## 概述

简历编辑器是一款简洁高效的在线简历制作工具，帮助您快速创建专业的中文简历。通过简单的表单填写，即可实时预览简历效果，并一键生成格式完美的PDF文件。

## 快速开始

### 1. 访问应用

打开浏览器，访问简历编辑器网址。您将看到主编辑界面，左侧是表单区域，右侧是实时预览区域。

### 2. 填写基本信息

在"基本信息"表单中填写：
- **姓名**：请输入您的真实姓名
- **邮箱**：请输入有效的邮箱地址
- **电话**：请输入您的联系电话
- **求职意向**：可选，输入您期望的职位

填写完成后点击"更新基本信息"按钮。

### 3. 添加教育背景

点击"教育背景"部分的"添加"按钮，填写：
- **学校名称**：输入您的毕业院校
- **专业**：输入您的专业名称
- **学历**：选择您的学历层次
- **开始时间**：选择入学时间
- **结束时间**：选择毕业时间（可选）
- **主修课程**：输入主要学习的课程（可选）

点击"添加"保存教育经历。您可以添加多条教育背景。

### 4. 添加实习经历

点击"实习经历"部分的"添加"按钮，填写：
- **公司名称**：输入实习公司名称
- **职位**：输入您的职位
- **开始时间**：选择实习开始时间
- **结束时间**：选择实习结束时间（可选）
- **工作描述**：详细描述您的工作内容和成就，每行一个要点

点击"添加"保存实习经历。您可以添加多条实习经历。

### 5. 添加个人技能

点击"个人技能"部分的"添加"按钮，填写：
- **技能类别**：如"编程语言"、"软件工具"、"语言能力"等
- **具体技能**：如"JavaScript"、"Excel"、"英语"等

点击"添加"保存技能信息。您可以添加多个技能条目。

## 实时预览

在您填写信息的同时，右侧预览区域会实时显示简历效果。您可以：
- 查看简历的整体布局
- 确认信息是否正确显示
- 调整内容以获得最佳视觉效果

## 生成PDF

### 预览完整简历

点击"预览完整简历"按钮，将在新页面中显示完整的简历预览。您可以：
- 查看最终效果
- 使用浏览器的打印功能
- 返回编辑页面继续修改

### 下载PDF文件

1. 点击"生成PDF"按钮
2. 系统将自动生成PDF文件
3. PDF文件将自动下载到您的设备

## 编辑和管理

### 修改已添加的信息

- 找到要修改的项目
- 点击编辑图标（铅笔图标）
- 修改相关信息
- 点击"更新"保存更改

### 删除信息

- 找到要删除的项目
- 点击删除图标（垃圾桶图标）
- 确认删除操作

### 重新排列信息

目前系统会自动按时间顺序排列教育背景和实习经历，最新的经历会显示在最前面。

## 移动端使用

### 手机和平板电脑

- 应用会自动适配移动设备屏幕
- 表单和预览会重新排列以获得最佳体验
- 所有功能在移动设备上都可以正常使用

### 触摸操作

- 点击按钮和输入框
- 滑动查看长内容
- 使用设备键盘输入信息

## 常见问题

### PDF生成失败怎么办？

1. 检查网络连接是否正常
2. 确保浏览器允许下载文件
3. 尝试刷新页面后重新生成
4. 如果问题持续，请联系技术支持

### 简历格式不符合预期？

1. 检查所有必填字段是否已填写
2. 确保日期格式正确
3. 预览时检查整体布局
4. 调整内容长度以获得最佳效果

### 如何保存我的简历？

- 所有输入的信息会自动保存在浏览器本地
- 关闭浏览器后重新打开，信息仍然存在
- 建议定期生成PDF文件作为备份

### 支持哪些浏览器？

- Chrome 90及以上版本
- Firefox 88及以上版本  
- Safari 14及以上版本
- Edge 90及以上版本
- 手机和平板设备的现代浏览器

## 使用技巧

### 内容优化建议

1. **简洁明了**：保持内容简洁，突出重点
2. **量化成果**：用具体数字描述成就
3. **关键词优化**：使用行业相关的关键词
4. **格式一致**：保持日期和格式的一致性

### 布局优化

1. **合理长度**：避免某个部分过长
2. **平衡内容**：各部分内容保持相对平衡
3. **字体大小**：系统会自动调整字体大小
4. **分页处理**：长简历会自动分页

### 专业建议

1. **量身定制**：根据申请职位调整内容重点
2. **突出亮点**：将最重要的信息放在显眼位置
3. **避免错误**：仔细检查拼写和语法错误
4. **保持更新**：定期更新简历内容
